# Şafak Özkan — İçerik & Aksiyon Input Listesi (FAZ 0.3)

> Master plan Bölüm 0 Kural 5 gereği: her sayfaya Şafak'ın **gerçek deneyiminden**
> en az 1 özgün unsur girmeli (uydurma yasak). Aşağıdaki input alınmadan ilgili
> sayfalar `[ŞAFAK INPUT GEREKLİ]` olarak taslak kalır.
> Erdi'ye iletilmek üzere derlendi — 2026-07-06.

## 🔴 Blokerlar (bunlar olmadan ilgili sayfa yayınlanamaz)

1. **Danışan deneyimleri (3 anonim vaka)** — `/demartini-deneyimler` sayfası için.
   Gerçek seanslardan, isim/kişisel veri olmadan, Şafak'ın kendi anlatımıyla.
   ⚠️ Uydurma testimonial kesinlikle yasak (Bölüm 0 Kural 2 + YMYL).

2. **Klinik unvan durumu** — Şafak psikolog/psikoterapist unvanı taşıyor mu?
   Taşımıyorsa sitede "terapi / tedavi / hasta" kelimeleri KULLANILMAZ; yerine
   "çalışma / seans / danışan / farkındalık süreci" (Bölüm 3.2). Netleştir.

## 🟠 İçerik inputları (sayfa kalitesi için)

3. **Bio / özgeçmiş** — `/hakkinda` + Person schema için: Demartini'ye nasıl geldi,
   kaç yıldır uyguluyor, arka planı, yaklaşımı. Kendi cümleleriyle.

4. **Seans anlatımı** — `/demartini-seansi` için: bir seans gerçekte nasıl akıyor,
   süre, online/yüz yüze farkı, danışanın ne hissettiği. Şafak'ın gözlemi.

5. **Her Tier-2 sayfasına 1 özgün unsur** — seans gözlemi / anonim örnek / kendi
   metaforu. (Bölüm 7 şablonu adım 5). Sayfa başına 2-3 cümle yeterli.

## 🟡 Lisans & doğrulama

6. **Eğitim/lisans tarihi** — "Trained" lisansı veriliş tarihinden 2 yıl geçerli.
   Tarih alınıp **yenileme tarihi takvime** işlenecek. Dizinden düşme = tüm
   doğrulama sinyalleri tersine döner (Bölüm 3.1).

7. **[ŞAFAK AKSİYONU] Enstitü'den bireysel profil sayfası talebi** — şu an
   drdemartini.com/facilitators'ta düz metin listeleme var, kişisel URL yok.
   Bireysel profil açılırsa `sameAs` schema o URL'e güncellenir.

## 🟢 Medya (Faz 3 / GEO)

8. **YouTube videosu** — 2-3 dk "Demartini seansı nasıl geçer" videosu. Sayfaya
   embed + VideoObject schema. LLM'ler YouTube'u yoğun kaynak alıyor (Bölüm 10).
   Çekim brief'i Şafak'a iletilecek.

9. **Profesyonel foto** — `/hakkinda` + Person schema image için.
